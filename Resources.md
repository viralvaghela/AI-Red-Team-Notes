### Tools / Useful Links

- [ JailbreakDB ](https://jailbreakdb.com/) - A database of known jailbreaks and prompt injection techniques for various LLMs.
- [PromptFoo](https://promptfoo.dev/) - A tool for testing and evaluating prompts against LLMs.
- [ MitRE ATT&CK for LLMs ](https://attack.mitre.org/) - A framework for understanding and categorizing attack techniques against LLMs.
- [ OpenAI's Security Best Practices ](https://openai.com/security-best-practices) - Guidelines and recommendations for securing LLMs and AI systems.
